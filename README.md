## Featured projects

### [PaperPal RAG Assistant](https://github.com/zakilbaki/paperpal-rag-assistant)

Scientific document intelligence application for uploading PDFs, generating summaries, extracting keywords, comparing papers, and asking retrieval-grounded questions over scientific documents.

Includes PDF ingestion, section-aware chunking, MongoDB persistence, local vector search, cited answers, retrieval evaluation, FastAPI backend, Streamlit UI, Docker Compose, tests, and CI.

**Highlights:** evaluated RAG pipeline with cited answers, clear separation between MongoDB document storage and vector-based retrieval, and local-first deployment.

### [Velib Demand Forecasting](https://github.com/zakilbaki/velib-demand-forecasting)

End-to-end forecasting and MLOps system for next-hour bike availability in Paris.
Includes live API ingestion, PostgreSQL storage, time-aware features, MLflow tracking, explicit model promotion, FastAPI serving, Docker Compose, tests, and CI.

**Highlights:** test RMSE `2.82`, test R² `0.94`, and a reproducible local serving workflow.

### [CryoSens Analytics](https://github.com/zakilbaki/cryosens_analytics)

Interactive industrial sensor toolkit for analyzing rapid thermal variations, operating cycles, threshold sensitivity, and cross-sensor event coincidence. Built for transparent, human-in-the-loop engineering analysis.

**Highlights:** reusable Python package, notebook dashboards, synthetic example data, Parquet exports, tests, and an analyst-facing user guide.
