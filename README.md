## Featured projects

### [Velib Demand Forecasting](https://github.com/zakilbaki/velib-demand-forecasting)

End-to-end forecasting and MLOps system for next-hour bike availability in Paris.
Includes live API ingestion, PostgreSQL, time-aware features, MLflow tracking, explicit
model promotion, FastAPI serving, Docker Compose, tests, and CI.

**Highlights:** test RMSE `2.82`, test R2 `0.94`, reproducible local serving workflow.

### [CryoSens Analytics](https://github.com/zakilbaki/cryosens_analytics)

Interactive industrial sensor toolkit for rapid thermal variations, operating cycles,
threshold sensitivity, and cross-sensor event coincidence. Built for transparent,
human-in-the-loop engineering analysis.

**Highlights:** reusable Python package, notebook dashboards, synthetic example data,
Parquet exports, tests, and an analyst-facing user guide.

### [Industrial Defect Prediction](https://github.com/zakilbaki/industrial-defect-prediction)

Leakage-aware baseline for early defect detection on an automotive production line.
Handles rare defects, missing process measurements, chronological validation, and
past-only rolling features.

**Highlights:** class-imbalance metrics, reproducible training CLI, explicit discussion
of metric integrity, tests protecting chronology and feature construction.
